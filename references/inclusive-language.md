# Inclusive language reference

This file contains the full inclusive language tables referenced by the brand voice skill.

## Ableist language

| ❌ Avoid | ✅ Use instead |
|---|---|
| crazy, insane | unheard of, surprising, intense |
| nitty-gritty | details, specifics |
| blacklist / whitelist | blocklist / allowlist |
| blind spot | gap, oversight |
| tone deaf | out of touch, unaware |

## Gendered language

| ❌ Avoid | ✅ Use instead |
|---|---|
| guys | folks, people, everyone, team |
| mankind | humankind, people |
| manpower | workforce, staffing |
| chairman | chair, chairperson |
| stewardess | flight attendant |

## Pronouns

- Default to **they/them** when a person's pronouns are unverified
- Mirror the pronouns used by the person you're writing about or for

## Racial & cultural language

| ❌ Avoid | ✅ Use instead |
|---|---|
| blacklist / whitelist | blocklist / allowlist |
| master / slave (tech) | primary / replica, leader / follower |
| grandfathered | legacy, exempt |
| cakewalk | straightforward |

## Capitalization

- Capitalize **Black**, **Native**, **Indigenous**
- Follow style guides for other ethnicity/nationality terms

## Disability language

- Webflow defaults to **identity-first** language (e.g., "disabled person") unless someone specifies a preference for person-first language (e.g., "person with a disability")
